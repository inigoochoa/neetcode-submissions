# NeetCode Solutions — @inigoochoa
Repositorio destinado a almacenar mis soluciones de NeetCode.
Este repo es un registro en curso, no una guía pulida — algunas soluciones antiguas pueden revisarse con el tiempo.

## Lenguajes

1. Python
2. SQL


## Estructura

Soluciones organizadas por carpeta de tema, luego por ID de problema. Cada submission se almacena como un archivo separado:

```
<carpeta-tema>/
  <id-problema>/
    submission-0.<ext>   ← primera submission
    submission-1.<ext>   ← segunda submission
    ...
```
